Tank game rules

Overview

We have a 10x10 grid with cells. The game is turn based. Every turn, each tank can perform one of the following actions:
Move/shoot up/right/down/ left
Moving = changing the current position of the tank (a tank can only move to a neighboring cell) Shooting = covering a portion of a row/column with a deadly laser, that kills every tank it touches

How to play

Players create a Python script that will control their tank. They then upload their scripts to a server. At the end of the day, there is a showdown battle between all the players to see who programmed the best AI

Structure of the script

def play_turn(grid):
do_something()
return action_code

Where grid is a 2D array of integers (the state of the board after the previous turn ended and:

  • grid[i][j] = 0 if the cell (i, j) is empty
  • grid[i][j] = 1 if the cell (i, j) has a tank
  • grid[i][j] = 2 if the cell (i,j) has a laser

And action_code is an integer representing the action to perform on the next turn. For example, 0 = Move up, 5 = Shoot right, etc
